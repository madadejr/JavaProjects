O Webmail da as boas vindas

Esta e uma aplicacao web que lhe permite fazer envio de emails para, onde para tal e necessario que se cadastre e fazer login com as credenciais.

A aplicacao tem um funcionamento bastante simples, a saber:

1. Para cadastro, insira as informacoes como os campos sugerem e e clique em Registar. Clique AQUI para o cadastro.

2. Para entrar (login) no sistema, devera digitar as credenciais nos campos e clicar em Entrar. Clique AQUI para entrar.

3. Para enviar o email, basta preencher devidamente os campos e clicar em Enviar.

NOTA: A tela do login permite-lhe deslocar-se para registo caso nao tenha conta!
Apos o cadastro, sao mostradas as informacoes gravadas e, a partir do link "Continuar", e direcionado a tela do login.
A partir da imagem no cabecalho pode ser direcionado para a Pagina Inicial.
